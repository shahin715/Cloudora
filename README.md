# Cloudora Website

A modern corporate website built using Next.js and Tailwind CSS.

##  Live Demo
https://cloudora-seven.vercel.app

##  Features
- Multi-page website (Home, About, Services, Career, News)
- Dynamic News Details Page
- API-based content rendering (JSON)
- Responsive design for all devices
- Smooth animations using AOS

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- JavaScript
- AOS (Animation on Scroll)

## Project Structure
- /src → components & pages
- /public → images & JSON data
